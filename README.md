# Portfolio — Neyla Haroune

Site portfolio Next.js 14 + TypeScript + Tailwind CSS + Framer Motion.

## Installation

```bash
npm install
npm run dev
```

Ouvre http://localhost:3000

## Déploiement sur Vercel

### Option 1 — Via GitHub (recommandé)
1. Push ce dossier sur GitHub
2. Va sur [vercel.com](https://vercel.com) → New Project
3. Importe ton repo GitHub
4. Vercel détecte Next.js automatiquement → Deploy !

### Option 2 — Via CLI
```bash
npm install -g vercel
vercel
```

## Personnaliser tes projets

Édite le fichier `data/projects.ts` pour modifier/ajouter des projets.

## Ajouter des vraies images pour Yemma / Circuits

Remplace dans `public/images/` :
- `yemma.png` → screenshot de ton app Flask
- `circuits.png` → capture Logisim

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icônes)
